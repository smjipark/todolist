## 📌 소개

별도 설치 없이 브라우저에서 바로 사용할 수 있는 할 일 관리 앱입니다.  
할 일에 설명을 추가하고, 그룹으로 분류하며, 완료 시간을 자동으로 기록합니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 할 일 추가 | 제목 + 설명(선택)을 입력하여 할 일 등록 |
| 그룹 관리 | 사이드바에서 원하는 이름의 그룹 생성 · 삭제 |
| 그룹 지정 | 할 일 등록 시 그룹 선택 (미선택 가능) |
| 그룹 필터링 | 그룹 클릭으로 해당 그룹 할 일만 표시 |
| 상태 필터 | 전체 / 진행 중 / 완료 기준으로 필터링 |
| 완료 처리 | 체크박스로 완료 표시, 완료 시각 자동 기록 |
| 선택 삭제 | 여러 항목을 선택 후 한 번에 삭제 |
| 전체 선택 | 헤더 체크박스로 보이는 항목 전체 선택 |

## 🚀 시작하기

### 실행 방법

별도 설치 없이 `index.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.

```
ToDoList/
├── index.html   # 메인 페이지
├── style.css    # 스타일
├── script.js    # 기능 로직
└── README.md
```

### 사용 방법

1. **할 일 추가**
   - 입력창에 제목 입력 → 폼이 확장되면 설명 · 그룹 선택 후 `추가` 클릭
   - 설명과 그룹은 선택 사항이며, Enter 키로도 빠르게 추가 가능

2. **그룹 만들기**
   - 왼쪽 사이드바 `그룹` 섹션의 `+` 버튼 클릭
   - 원하는 이름 입력 후 `저장` 또는 Enter

3. **그룹으로 보기**
   - 사이드바에서 그룹 이름 클릭 → 해당 그룹 할 일만 표시
   - 같은 그룹 재클릭 시 필터 해제

4. **완료 처리**
   - 항목 왼쪽 체크박스(완료용) 클릭 → 완료 처리 + 시각 기록

5. **선택 삭제**
   - 빨간 체크박스로 항목 선택 후 `선택 삭제` 버튼 클릭

## ⚠️ 데이터 저장 안내

현재 버전은 **브라우저 메모리에만 데이터를 저장**합니다.

- 페이지를 **새로고침하거나 탭을 닫으면 모든 데이터가 초기화**됩니다.
- 데이터를 유지하려면 탭을 닫지 말고 그대로 유지해 주세요.
- 영구 저장 기능(localStorage)은 향후 버전에서 추가될 예정입니다. → [로드맵](#️-로드맵) 참고

## 📁 파일 구조

```
index.html
├── 상단 헤더 (제목, 버전, 상태 필터 버튼)
└── 메인 레이아웃
    ├── 그룹 사이드바
    │   ├── 그룹 추가 입력창
    │   └── 그룹 목록 (동적 렌더링)
    └── 콘텐츠 영역
        ├── 입력 카드 (인라인 확장 폼)
        └── 할 일 목록 (동적 렌더링)

script.js
├── 그룹 관리: addGroup / deleteGroup / setGroupFilter / renderGroups
├── 입력 폼:   expandForm / collapseForm / addTodo
└── 할 일:     toggleTodo / toggleSelect / deleteSelected / render

style.css
├── 레이아웃: 상단 헤더, 그룹 사이드바, 콘텐츠
├── 컴포넌트: 입력 카드, 목록 섹션, 할 일 아이템
└── 상태:     done, dimmed, selected
```
